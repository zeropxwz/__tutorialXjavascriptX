# JavaScript turorial

## Переменные 

Для объявления переменных и констант используются ключевые слова var, let и const

### декларация переменных

Объявить переменные можно несколькими способами:  

```js
var a
var b
var c
```

```js
var a, b, c
```

```js
var 
    a, 
    b, 
    c
```
### инициализация переменных

```js
a = 2
b = 10
c = null
```
